## 1. ПРАВИЛА МОДЕРАЦИИ TIER-III

### 1.1. Общие положения

* **1.1.1** ▸ Получив права Tier-III, вы автоматически принимаете на себя ответственность за соблюдение данных правил.
* **1.1.2** ▸ Незнание правил модерации не освобождает от ответственности.
* **1.1.3** ▸ Администрация вправе корректировать полномочия Tier-III без предварительного уведомления.
* **1.1.4** ▸ Модераторы Tier-III подчиняются администрации и старшим модераторам.
* **1.1.5** ▸ Все действия модератора должны соответствовать общей политике TeeFusion.

### 1.2. Запрещенные действия (Модератор)

| Пункт | Нарушение | Детали | Наказание |
| :--- | :--- | :--- | :--- |
| **2.1** | **Неуважительное обращение с игроками** | Оскорбления, провокации, унижения, предвзятое отношение. | Выговор, при повторном нарушении — **Предупреждение** |
| **2.2** | **Игнорирование обращений** | Отказ проверять подозрительную активность по запросу или неподчинение указаниям старших. | Предупреждение, при систематическом нарушении — **Временное отстранение** |
| **2.3** | **Злоупотребление полномочиями** | Использование багов/эксплойтов, абуз административных команд, использование режима *super* (годмода) для получения игрового преимущества. | **Предупреждение** |
| **2.4** | **Нарушение игрового баланса** | Чрезмерно частое проведение турниров, телепортация игроков без их согласия (кроме случаев нарушений). | Выговор, при повторном нарушении — **Временное отстранение** |
| **2.5** | **Необоснованные обвинения** | Ложные обвинения в читерстве без доказательств, распространение непроверенной информации. | **Выговор**, при тяжком нарушении **Временное отстранение** |
| **2.6** | **Проявление фаворитизма** | Предвзятое отношение к друзьям/знакомым, игнорирование нарушений определенных игроков. | Выговор, при повторном нарушении — **Предупреждение** |

### 1.3. Обязанности модератора

* **Контроль нарушений (3.1):**
    * При обнаружении читера: Выдать бан согласно регламенту и отчитаться в канале логов.
    * При злоупотреблении другим модератором: Собрать доказательства, немедленно сообщить администрации, не предпринимать самостоятельных действий.
    * При использовании багов игроками: Записать демонстрацию экрана, выдать соответствующее наказание и сообщить о баге разработчикам.
* **Контроль специальных режимов (3.2):**
    * **Block режим:** Следить за корректностью блоков, предотвращать троллинг.
    * **KoG режим:** Обеспечивать очередность на популярных картах, пресекать намеренные помехи, следить за ротацией на ограниченных слотах.
    * **Тренировочные зоны:** Обеспечивать равный доступ, помогать новичкам по запросу, предотвращать AFK-блокировку.
* **Коммуникация и поддержка (3.3):**
    * Оперативно реагировать на жалобы игроков.
    * Помогать в решении технических проблем.
    * Разъяснять правила при необходимости.
    * Поддерживать позитивную атмосферу.
* **Отчетность (3.4):**
    * При длительном отсутствии — уведомить в канале **"Отпуск"**.
    * При обнаружении ботов — сообщить в **"Модерация" с тегом**.
    * Вести журнал принятых решений по спорным ситуациям.
    * Регулярно просматривать логи сервера.

### 1.4. Права и полномочия Tier-III

* **Использование команд (5.1):** `kick`, `ban`, `mute`, `tele` для разблокировки игроков.
* **Доступ к информации (5.2):**
    * Просмотр IP-адресов для выявления мультиаккаунтов.
    * Доступ к логам чата за последние 24 часа.
    * Просмотр истории нарушений игроков.
* **Экстренные полномочия (5.3):** Ротация игроков между режимами при перегрузке, немедленный бан при явных нарушениях.

### 1.5. Формат наказаний

> **Стандартный формат:** `ban <ip> <время> <"ник нарушителя/причина/ник модера">`

* **1.** ▸ При возникновении спорных ситуаций, не описанных в правилах, решение принимается **коллегиально** с участием старшей модерации.
* **2.** ▸ Модератор Tier-III является представителем TeeFusion и должен поддерживать **репутацию проекта** своим поведением.
* **3.** ▸ Активная и качественная работа может быть поощрена повышением до старшего модератора.
